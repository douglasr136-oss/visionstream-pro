Instruções do VISIONSTREAM PRO

1 Ativação do Sistema

Informe um endereço MAC e uma chave de acesso válidos para ativar o sistema. Use o formato correto para ambos.

2 Carregar Lista IPTV

Cole a URL da sua lista M3U no campo fornecido e clique em "Carregar". Você também pode usar uma das listas de exemplo.

3 Navegar nos Canais

Após carregar a lista, os canais serão exibidos. Clique em qualquer canal para iniciar a reprodução.

4 Recarregar Lista

Use o botão "Recarregar Lista" para atualizar a lista de canais com a URL atual.

Notas Importantes

  • Este player funciona apenas com listas M3U públicas via HTTP/HTTPS
  • Alguns canais podem não funcionar devido a restrições CORS ou formato não suportado
  • Para melhor experiência, use o aplicativo em dispositivos compatíveis com HLS
  • Este é um player web e depende da compatibilidade do navegador
